# JS Assignment – Interactive Web Pages with JavaScript

A React + Vite project covering all 5 parts of the JavaScript assignment.

## Parts Covered
1. JavaScript Basics – variables & console output
2. DOM Manipulation – change heading on button click
3. Event Handling – addEventListener click logger
4. Simple Calculator – addition, subtraction, multiplication, division
5. To-Do List – add, remove, mark complete (bonus: progress bar)

## How to Run

### Prerequisites
Make sure you have **Node.js** installed → https://nodejs.org (download LTS)

### Steps

```bash
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev

# 3. Open in browser
# Visit: http://localhost:5173
```

### Build for Production
```bash
npm run build
npm run preview
```
